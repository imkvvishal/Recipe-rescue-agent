Recipe Rescue Agent

Recipe Rescue Agent is an AI-powered web application that generates personalized recipes using available ingredients. The system utilizes four specialized AI agents to provide cooking instructions, allergy-aware recommendations, nutritional insights, and food waste reduction suggestions.

Features

* Generate recipes from available ingredients
* Step-by-step cooking instructions
* Allergy-aware ingredient recommendations
* Nutritional analysis and suggestions
* Food waste reduction guidance
* Powered by Google Gemini AI

Tech Stack

* React.js
* TypeScript
* Vite
* Node.js
* Express.js
* Google Gemini AI

AI Agents

Instruction Agent – Generates cooking steps
Allergy Agent – Detects allergens and suggests alternatives
Nutrition Agent – Provides nutritional insights
Waste Reduction Agent – Minimizes food wastage
